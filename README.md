<p>“我”回过神来发现自己身处于一座树屋内。<br>
出口在何处？<br>
那小女孩和老爷爷是什么人？<br>
我为何会在这里？</p>
<p>《Treehouse Riddle》是一款通过解开顿悟谜题与逻辑谜题来推进故事进程的<br>
“顿悟谜题×逻辑谜题 冒险游戏”。</p>
<p>主角被困在了树屋的一间房内。<br>
在用怀旧像素风格绘制的房间内探索，<br>
通过解开谜题来寻找逃离树屋的手段。</p>
<p>正如同树屋一般，谜题由温馨的木制纹理绘制。<br>
一起来不紧不慢地体验总数超过60个的智力挑战吧。</p>
<p>所有谜题都拥有“提示”和“跳过”功能。</p>
